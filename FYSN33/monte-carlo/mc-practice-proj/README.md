# Monte Carlo Practise — Lecture 2

This repository contains the **Workshop practice notebook** for the *Applied Computational Physics and Machine Learning* course at Lund University, FYSN33.

## 📘 Overview

The notebook introduces and extends the Monte Carlo sampling methods discussed in Lecture 2 (*Advanced Sampling*).  
You will implement and test several algorithms, including:

- Inversion sampling  
- Hit–miss (accept/reject) sampling  
- Overestimates and envelope functions  
- Accept-all with weights (`w = f/g`)  
- Event reweighting for biased coin tosses  

The exercises build directly toward the *Entropic Spring* project.
